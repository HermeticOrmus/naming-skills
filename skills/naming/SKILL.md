# Naming

## Principle

Mine the brand's vocabulary first. Invent only when the corpus is exhausted.

## Workflow

1. Understand product function (verb + parts + user relationship)
2. Mine vocabulary in source-preference order
3. Generate 10 candidates with source + meaning + register
4. Present 3-5 finalists, ask for taste pivot
5. Commit + document

## Scale to suite

Thematic root + role hierarchy + same root for every product.

## NOT for

Variable / file / function / flag names.
